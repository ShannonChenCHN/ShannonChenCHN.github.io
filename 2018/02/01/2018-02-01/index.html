2018-02-01

Feb 1, 2018

之前每次写文章都要先发到 GitHub 上，然后再同步到简书、掘金、公众号等多个平台，感觉真有些疲于奔命了，后来想想，公众号就没必要了。

先实践，再思考，写出来的东西才有意义。

真正的好文章不应该被条条框框所限制住，我个人最喜欢看的文章还是不拘泥、有个性、有思想的文风。在技术圈，最喜欢 bang、casa 和 bestswifter 的文章，因为他们用很直白、很的文字把复杂的技术讲的很清楚，逻辑思路都特别清晰，不拖泥带水，最重要的是能展现自己思想的独特性。

要说解决拖延症的最好办法，就是拆分的尽量细，然后在规定时间内集中精力一个一个干掉。
