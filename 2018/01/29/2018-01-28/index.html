2018-01-28

Jan 29, 2018

昨天读了韩帅分享的面经后，触动很大，今天上午把昨天没写完的面试总结写完了。感觉跟这些优秀的人比，自己差太多了。对比简历，其中很两点让我受益很深：

  • 写简历要清晰简洁，要点一看上去就很清楚，而不是看上去一大堆，让人觉得太多了，另外，就是一定简历要不断改进。其实，做其他事也一样，要抓住要点，不断改进。
  • 简历中有些让人眼前一亮的东西很重要。比如，“iOS 负责人兼前端工程师”、“10+个项目经验”、“virtual memory”技术分享、主导技术选型、“开源项目曾占领 github trending swift 榜两天”、小程序开发等等。

晚上，看了下湖南卫视新出的『声临其境』这个节目，看了韩雪的表演后，瞬间迷上了这个节目。
